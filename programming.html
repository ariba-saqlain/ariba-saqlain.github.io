3D Graphics Simulator | C/C++

Top

This program begins with an animation of a 3D cube being rotated. This was done by calling a function called move() which draws the image, waits 40 milliseconds, and then calls move() to redraw the image again. This approach was chosen to minimize the CPU utilization rate as much as possible.

A series of different transformations and animations are shown next:

  1. Model Transformations: The cube (model) is rotated with respect to its x, y, and z axis. It is also shown being scaled to smaller and larger sizes.
  2. WC (world coordinate) Transformations: The cube is translated along the world's x, y, and z axis.
  3. Simple Solar System: An animation is created by rotating the sun in the centre of WC, rotating the earth along its own z axis and around the sun, and rotating a smaller blue planet around earth. Next, bitmap images are mapped onto each planet to create the texture.
  4. Bezier Curve Generation: Through mouse clicks, control points are generated and displayed and these points are then used to generate the Bezier curve. This was done by computing the coefficient polynomials, using them with the control points to compute the point p(u) on the curve for any given value of u, and then drawing the Bezier curve through line segments.
  5. Cube Texture: Different texture mappings are applied to each face of this cube. It goes through transformations known as View, where the eye position is rotated with respect to the world's x, y, and z axis.

Animal Kingdom | C/C++

Top

  • A right-click menu gives users the option to choose between six different 2D and 3D animals.
  • Some animals have two further options:
  • Animation: where the animal is shown transforming by itself.
  • User Controlled: where the user can control the transformation.
  • Next, the user selects an action for the animal to perform such as jump, run, swim, etc.
  • Once this selection is made, the animal is shown in its own natural habitat performing the action chosen by the user.
  • All animals can be seen performing the jump action if the user selects All Animals Jump.
  • To view the animals in the day or night, there is also an option to change the time of day.
  • Finally, the user may select Quit to exit the program.

Calculator | Java

Top

  • This program was made by using a graphical user interface to create buttons, assign values, and display the numbers as they are pressed. Formulas were then added to calculate the result, and situations where the user enters invalid input are handled.
  • Expected sequence of actions:
  • Enter the first number
  • Enter an operator / * - +
  • Enter the second number
  • Enter the operator =
  • Enter the operator C (to clear the screen and reset the calculator)
  • If the user enters an invalid sequence of actions, the program will output 0.0
  • The video shown above demonstrates the following situations:
  • Division without a remainder
  • Division with a remainder
  • Multiplication
  • Subtraction resulting in a position answer
  • Subtraction resulting in a negative answer
  • Addition
  • Invalid sequence of actions
